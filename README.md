# Crypto Staking and Rewards (Web3)

This project simulates a basic crypto staking system.

It helps beginners understand how users lock tokens and earn rewards over time.

## What this project does

- Stakes tokens for users
- Tracks staking balances
- Calculates rewards
- Shows total balance with rewards

## Why this project is useful

You will learn:
- How staking works
- How reward logic is written
- How balances are tracked
- How blockchain systems are structured

## Tech used

- JavaScript
- Node.js
- Staking concepts
- Blockchain logic (simulated)

## How to run

1. Install Node.js  
2. Clone this repo  
3. Open terminal in project folder  
4. Run: `node app.js`  

## Files overview

- `app.js` → Main app file  
- `stake.js` → Staking logic  
- `reward.js` → Reward calculator  
- `user.js` → User validation  
- `pool.js` → Staking pool  
- `storage.js` → Data storage  
- `config.js` → Settings  
- `logger.js` → Logs  
- `sample.env` → Config example  
- `package.json` → Project info
